# 코맥스 월패드 애드온 (mqtt)
## CommaxWallpadBySaram
mqtt를 이용한 코맥스 월패드 애드온. ew11 같은 장비가 필요합니다.
- 리빌딩 (이라고 하지만 이전꺼와 거의 유사합니다.)
- 전열기가 없어서 테스트가 안됩니다.
- 엘리베이터는 제가 안쓰다 보니 테스트를 안합니다.


## 20240710 추가 수정
- RS485 특성상 많은 데이터가 오가면 원하는 데이터 송신시 작동안함.
- 데이터 Send시 데이터를 큐에 5개씩 넣고 순차 전송.
