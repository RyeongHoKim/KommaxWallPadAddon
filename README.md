# 코맥스 월패드 애드온 (mqtt)
## CommaxWallpadByTcT
mqtt를 이용한 코맥스 월패드 애드온. ew11 같은 장비가 필요합니다. (EW11 커뮤니케이션 MQTT Mode)
- 리빌딩 (사람님 코드 가져와서 수정.)
- RS485 특성상 많은 데이터가 오가면 원하는 데이터 송신시 작동안함 -> 작동후 Recv 받을때까지 Send함.
- 엘리베이터 확인중.
- 전열교환기 확인중.
